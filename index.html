Xbox 360 controller on cubase

Well i found a way to use our Xbox controller in cubase i tested in cubase elements 8; so we're gonna need the following softwares

  1. GlovePIE
  2. LoopMidi / Loopbe1

Simply download and install those programs and download the repository.

Search the .PIE file and once there open it with GlovePIE

Configure manually the midi output and channel, finally press Run

Open Cubase and create a new Key Command for the Monitor button of the control; the command will be Ctrl+Shift+M
